# Terminal oh-my-zsh setup

Enhance your terminal with useful information like current branch, commit update and git stashes can significantly speed up your development process. In this guide, we will transform your Mac OS terminal into an insightful yet stylish interface using [**Oh My Zsh**](https://ohmyz.sh/).

![Mac OS terminal customized with oh-my-zsh](/img/491cbd5e-9c11-43f8-8e4c-26ff91773b89.png)

1. **Installing [**Oh My Zsh**](https://ohmyz.sh/) using curl**

   To get started, open your terminal and run the following command:

   ```
   sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
   ```

If the installation was successful, you will be greeted with a welcome UI from Oh My Zsh.
